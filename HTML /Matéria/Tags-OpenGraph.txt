		Tags Open Graph

 Tags Open Graph são o protocolo que nos permite controlar como o conteúdo de nosso site aparece nas várias midias sociais como o X ou qualquer outro producto da Meta.
 Ao definir essas propriedades, podemos atrair usuarios para que interajam com nosso conteúdo, estas podem ser definidas por meio de uma coleção de elementos meta, na secção <head> do documento HTML.
 
 <meta property="og:title" content="Seu título"/> Define o título usado nas postagens de nossa página em redes sociais.
 
 <meta property="og:type" content="Website"/> Define o tipo de conteúdo que estamos a postar.
 
 <meta property="og:img" content="link para imagem"/> Define a imagem que será exibida dentro dessa mesma postagem.
 
 <meta property="og:URL" content="https://link.com/> Define a URL que será usada para as postagens contendo determinada página web, a quando nosso usuario clicar.
