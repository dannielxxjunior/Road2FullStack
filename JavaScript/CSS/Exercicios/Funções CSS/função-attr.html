Função attr()

A função attr() é usada para acessar o valor de um atributo de um elemento HTML. Ela é comumente utilizada em CSS para aplicar estilos com base nos atributos dos elementos.

Exemplo de uso

Suponha que temos o seguinte HTML:

    <div class="box" data-color="red">Box 1</div>
    <div class="box" data-color="blue">Box 2</div>

Podemos usar a função attr() para definir a cor de fundo das caixas com base no valor do atributo data-color:

    .box {
      width: 100px;
      height: 100px;
      background-color: attr(data-color);
    }

No exemplo acima, a caixa com o atributo data-color="red" terá um fundo vermelho, enquanto a caixa com o atributo data-color="blue" terá um fundo azul.

Outros Exemplos de uso

Podemos usar a função attr() para exibir uma URL de um link, fazendo uso da mesma em conjunto com a propriedade content: e o pseudo-elemento ::before ou ::after.

    <a href="https://www.example.com">Example</a>

    a::after {
      content: " (" attr(href) ")";
    }

O resultado será um link com a URL exibida após o texto.
