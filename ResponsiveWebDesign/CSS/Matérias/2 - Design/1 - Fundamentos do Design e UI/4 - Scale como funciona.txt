			Como Scale funciona no Web Design
			
A "escala" de algo refere-se a seu tamanho.

Quando estamos analisando a escalabilidade de nosso design web, estamos observando as relações de tamanho entre diferentes elementos e como esses elementos podem se adaptar a diferentes tamanhos de tela.

Usar a escala correta para nossos elementos desempenha um papel importante na hierarquia visual. Elementos maiores chamarão mais a atenção, o que pode guiar seus usuários pelo conteúdo da maneira que você deseja.

Por exemplo, a separação visual entre um título e um parágrafo chama a atenção do leitor, mas a escala deve ser apropriada para obter um texto atraente que leve o leitor para essa secção.

	  }

	  .paragraph {
		font-size: 16px;
		max-width: 600px;
		line-height: 1.6;
	  }

	<div class="section">
