			Comon criar um bom contraste entre o plano de fundo e o primeiro plano em nossos Designs?
			
Primeiro que tudo vamos familiarizar-nos com o contraste. Contraste é a diferença entre duas cores - ou o quau fácil é distingui-las.

Cores com maior contraste serão mais visualmente distintas, enquanto cores com menor contraste serão mais visualmente semelhantes. Por exemplo, preto e branco têm uma relação de contraste muito alta. Enquanto o azul claro e o roxo claro têm menos contraste.

	Exeperimenta executar este proximo exemplo:
	
HTML

	<style>
	  .example {
		padding: 20px;
		margin-bottom: 20px;
		font-size: 18px;
		font-family: sans-serif;
	  }

	  .high-contrast {
		background-color: black;
		color: white;
	  }

	  .low-contrast {
		background-color: #add8e6; 
		color: #dda0dd; 
	  }

	  .label {
		font-weight: bold;
		margin-bottom: 5px;
	  }
	</style>

	<div class="example high-contrast">
	  <div class="label">High Contrast</div>
	  This text has high contrast (black background and white text).
	</div>

	<div class="example low-contrast">
	  <div class="label">Low Contrast</div>
	  This text has low contrast (light blue background and light purple text).
	</div>
	
Nestes exemplos temos ainda assim uma distinção mais clara por conta do layout. 

Mas como determinamos o que é um contraste "bom o suficiente"?
Não podemos basear isso apenas em como o texto nos parece, pois cada usuário terá uma experiencia diferente. Felizmente, Diretrizes de Acessibilidade para Conteúdo Web, ou WCAG(Web Content Accessability Guidelines), fornecem um padrão para isso.

Texto com uma proporção de contraste de 4.5:1 é considerado padrão AA, que é o mínimo necessário que devemos seguir para ser acessível para a maioria dos usuários. Texto com uma relação de contraste de 7:1 é considerado o padrão AAA e garante o melhor nível de acessibilidade.

Existem vários sites que podem verificar a relação de contraste entre duas cores, mas a maioria dos navegadores permite que você faça isso diretamente nas ferramentas de desenvolvedor do seu site.

Vamos em seguida ver alguns exemplos de Contrast Ratio para entendermos melhor o conceito.

Começamos com um exemplo de texto preto em fundo branco que tem uma relação de contraste de 21:1 :

		display: block;

	<div class="contrast-21">
	  <span class="label">Contrast Ratio 21:1</span>
	  This is black text on a white background, which has the highest contrast ratio of 21:1.
	</div>
	
Este exemplo atende mais do que o padrão AAA. Agora vamos dar uma olhada no texto roxo em um fundo azul.

Este exemplo tem uma relação de contraste de 1.48:1 que nem sequer atende ao padrão AA.

Como podemos corrigir isto? Bem há três aspetos que impactam a relação de contraste.

O primeiro é o Matiz, que representa o tipo geral de cor, como vermelho, azul elaranja. Vamos mudar para um Matiz que esteja mais distante do azul. Neste caso iremos usar o vermelho.

HTML:
	<style>
	  .contrast-21 {
		background-color: white;
		color: black;
		padding: 15px;
		font-family: sans-serif;
		font-size: 18px;
		margin-bottom: 20px;
	  }

	  .label {
		font-weight: bold;
		margin-bottom: 8px;
		display: block;
	  }

	  .red-on-blue {
		background-color: #0000cc; 
		color: #ff0000; 
		padding: 15px;
		font-family: sans-serif;
		font-size: 18px;
		margin-bottom: 20px;
	  }
	</style>

	<div class="contrast-21">
	  <span class="label">Contrast Ratio 21:1</span>
	  This example more than meets the AAA standard with black text on white background.
	</div>

	<div class="red-on-blue">
	  <span class="label">Red on Blue (Higher Contrast Hue Shift)</span>
	  This doesn't meet accessibility standards.
	</div>

Infelizmente, essa alteração trouxe a relação de contraste apenas para 1,49:1,o que significa que teremos que alterar a saturação ou a "quantidade" de cor presente. Vamos aumentar a quantidade de vermelho no texto. Isso nos aproxima muito mais do objetivo, com uma relação de contraste de 3,54:1.

	style>
	  .contrast-21 {
		background-color: white;
		color: black;
		padding: 15px;
		font-family: sans-serif;
		font-size: 18px;
		margin-bottom: 20px;
	  }

	  .label {
		font-weight: bold;
		margin-bottom: 8px;
		display: block;
	  }

	  .low-sat-red-on-blue {
		background-color: #0000cc; 
		color: #b23333; 
		padding: 15px;
		font-family: sans-serif;
		font-size: 18px;
		margin-bottom: 20px;
	  }

	  .high-sat-red-on-blue {
		background-color: #0000cc; 
		color: #ff4d4d;
		padding: 15px;
		font-family: sans-serif;
		font-size: 18px;
		margin-bottom: 20px;
	  }
	</style>

	<div class="contrast-21">
	  <span class="label">Contrast Ratio 21:1</span>
	  This example more than meets the AAA standard with black text on white background.
	</div>

	<div class="low-sat-red-on-blue">
	  <span class="label">Low Saturation Red on Blue (Contrast ~1.49:1)</span>
	  This red has low saturation, resulting in a poor contrast ratio.
	</div>

	<div class="high-sat-red-on-blue">
	  <span class="label">Higher Saturation Red on Blue (Contrast ~3.54:1)</span>
	  Increasing the saturation of red improves contrast but it’s still below AA standard.
	</div>
	
No entanto, podemos potencialmente chegar ainda mais perto alterando o último valor, que é lightness. A claridade representa a quantidade de branco presente na cor.

Vamos diminuir a luminosidade do vermelho. Agora há um vermelho muito mais escuro contra o fundo azul claro, o que traz a relação de contraste para 10,34:1.

	  }

	<div class="contrast-21">
	  <span class="label">Contrast Ratio 21:1</span>

	  <span class="label">Low Saturation Red on Medium Blue (Contrast ~1.49:1)</span>

	<div class="high-sat-red-on-blue blue-bg">
	  <span class="label">Higher Saturation Red on Medium Blue (Contrast ~3.54:1)</span>
	  Increasing the saturation of red improves contrast but it’s still below AA standard.
	</div>

	<div class="dark-red-on-light-blue">
	  <span class="label">Darker Red on Light Blue (Contrast ~10.34:1)</span>
	  Decreasing the lightness of the red increases the contrast ratio significantly.
	  
Podemos continuar ajustando as cores para encontrarmos um equilibrio entre o efeito visual que desejamos e uma relação de contraste acessível. Mas é importante lembrar que a acessibilidade deve sempre ter precedência.
