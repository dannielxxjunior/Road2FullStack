HTML5 Tables

The HTML <table> element allows Web Authors to display tabular data (such as text, images, links, other tables, etc.) in a two dimensional table with Rows and Columns of Cells.

Example one

This is our first simple table example

This will render a <table> consisting of three total rows (<tr>): one row of header cells (<th>) and two rows of content cells (<td>). <th> elements are tabular headers and (<td>) elements are tabular data.

You can put whatever you want inside a <td> or <th>

The Code

          
            <table>
              <tr>
                <th>Heading 1/Column 1</th>
                <th>Heading 2/Column 2</th>
              </tr>
              <tr>
                <td>Row 1 Data Column 1</td>
                <td>Row 1 Data column 2</td>
              </tr>
              <tr>
                <+<td>Row 2 Data Column 1</td>
                <td>Row 2 Data Column 2</td>
              </tr>
            </table>

The Result

Heading 1/Column 1 Heading 2/Column 2
Row 1 Data Column 1 Row 1 Data column 2
Row 2 Data Column 1 Row 2 Data Column 2

Example 2

Table cells can span multiple columns or rows using the colspan and rowspan atributes.

These atributes can be applied to <th> and <td> elements.

Note that you should not design a table where both rows and columns overlap as this is invalid HTML and the result is handled differently by different web browsers.

rowspan = A non-negative integer that specifies the number of rows spanned by a cell.

colspan = A non-negative integer that specifies the number of rows spanned by a cell.

The Code

          
            <table>
              <tr>
                <td>row 1 col 1</td>
                <td>row 1 col 2</td>
                <td>row 1 col 3</td>
              </tr>
              <tr>
                 This second row spans all three columns</td>
              </tr>
              <tr>
                <td rowspan="2">This cell spans for 2 rows</td>
                <td>row 3 col 2</td>
                <td>row 3 col 3</td>
              </tr>
              <tr>
                <td>row 4 col 2</td>
                <td>row 4 col 3</td>
              </tr>
            </table>

The Result

row 1 col 1 row 1 col 2 row 1 col 3
This second row spans all three columns
This cell spans for 2 rows row 3 col 2 row 3 col 3
row 4 col 2 row 4 col 3
