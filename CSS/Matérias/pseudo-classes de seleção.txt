			O que são :first-child e :last-child?

São pseudo-classes CSS que te permitem selecionar elementos com base na sua posição dentro de um elemento pai.	

Importante enteder que o uso destas pseudo-classes conta sempre a posição vista a partir do elemento PAI do elemento referenciado na pseudo-class.

	:fisrt-child
	
 •Seleciona o primeiro filho de um elemento pai.
 
Exemplo:

HTML
	<ul>
	  <li class="item">Item 1</li>
	  <li class="item">Item 2</li>
	  <li class="item">Item 3</li>
	</ul>

CSS
	.item:first-child {	
		color: red;
	}
	
Resultado:
	Apenas o primeiro item da lista será colorido a vermelho.
	
Usando a pseudo class :last-child teriamos o mesmo resultado mas no último item da lista.

Ou seja: 
	O uso destas pseudo-classes permite-nos aprlicar propriedades especificas a determinado elemento dentro de uma class, mesmo que façamos uso dessa classe em varios elementos, podemos assim selecionar apenas um dentro desses tantos.
---------------------------------------------------------------------------------

			Regra Importante
			
Ao trabalharmos com este tipo de pseudo-class é importante entender que as mesmas se referem exatamente ao primeiro e ultimo filhos do elemento em questão, sejam eles todos o mesmo tipo de elemento, ou não.

Por Exemplo:
	Usando a pseudo-class no seguinte HTML, que elemento será estelizado?
	O elemento <p>, pois é o primeiro filho da <div> para a qual a nossa pseudo-class aponta.

HTML
	<div>
	  <p>Texto</p>
	  <li>Item 1</li>
	  <li>Item 2</li>
	</div>
	
CSS
	li:first-child {
	  color: red;
	}
	
Mas podemos usar uma pseudo-class que seleciona o primeiro ou último filho do tipo que desejarmos, usamos para esse efeito as pseudo-class's 

	• :first-type-of 
	• :last-type-of

Exemplo:
	li:first-type-of {
		color:red
	}
--------------------------------------------------------------------------------

			:nth-child()
			
Até aqui já entendemos como fazemos para selecionar o primeiro ou o último elementos de entre vários sem que para isso precisemo atribuir uma class diferente a cada um, ou fazendo uso de ID's, mas e se quisermos selecionar especificamente o terceiro, ou o quarto elemento dentro de uma lista?

Para isso fazemos uso da pseudo-class :nth-child()
Esta importante pseudo-class peermite-nos selecionar qualquer elemento de entre vários, apenas precisamos saber qual o número do mesmo de entre os outros tantos, vejamos.

Exemplo:

HTML
	<ul>
	  <li>Item 1</li>
	  <li>Item 2</li>
	  <li>Item 3</li>
	  <li>Item 4</li>
	  <li>Item 5</li>
	</ul>
	
CSS
	li:nth-child(4) {
		color: purple;
	}
	
Simples assim.

Tal como com :first & :last-child com :nth também nos é possivel selecionar  partir do tipo do elemento, usando para isso :nth-of-type().

Exemplo: 
	li:nth-of-type(3)
